PROJECT: SantaDirectory.com / SantaDirectory.co.uk

OBJECTIVE:
Build a clean, modern, premium directory website for professional Santas for hire across the UK. This is NOT a booking engine or self-serve membership platform yet. For version 1, the site is a curated listing directory where visitors can search and browse Santa profiles, and potential Santas can contact us to request a listing. We will manually add and manage listings in the early stages. Do NOT build Stripe, login, member dashboard, or profile self-management yet. Structure the code so these can be added later.

IMPORTANT:
This is a polished MVP / brochure-directory build first.
Keep it simple, scalable, SEO-friendly, and visually strong.
Do not over-engineer.
Do not add fake complexity.

DESIGN DIRECTION:
- Premium, festive, modern, trustworthy
- Clean UI, not tacky or cheesy
- Festive styling should be subtle and elegant
- Think premium directory / marketplace design
- Mobile-first responsive layout
- High-quality card layouts and polished spacing
- Use a tasteful festive palette:
  - deep red
  - warm white / cream
  - dark green accents if suitable
  - soft gold accents very sparingly
- The site should feel more “premium Christmas brand” than “kids’ party leaflet”

SITE POSITIONING:
SantaDirectory is a trusted directory of professional Santas for hire.
Visitors can search by area and view polished Santa profile pages.
Santas who want to be listed can enquire through a form and we will help create their profile manually.

MVP FEATURES TO BUILD NOW:

1. HOMEPAGE
Build a premium homepage with:
- Hero section
- Strong headline:
  “Find a Professional Santa Near You”
- Subheading:
  “Browse trusted Santa performers for home visits, grottos, schools, and corporate events across the UK.”
- Search bar UI in hero (can be visual for now, but should link to listings page)
- CTA buttons:
  - “Find a Santa”
  - “List Your Santa Profile”
- Featured Santa cards section
- Trust/features strip with items such as:
  - Professional Santa performers
  - DBS-declared profiles
  - Real beard Santas available
  - Direct contact with performers
- “How it works” section:
  1. Search by area
  2. Browse Santa profiles
  3. Contact Santas directly
- Popular location links section
- CTA section for Santas who want to be listed
- Clean footer with navigation and placeholder policy links

2. LISTINGS PAGE

  - radius
  - DBS checked / DBS declared
  - real beard
  - home visits
  - corporate events
  - schools / nurseries

  - Santa name

  - location

  - short excerpt

- Layout should feel premium and easy to scan

The page should feel polished, premium, and SEO-friendly.
